## Description

This command can be used to activate the $5 Patreon perks on one server. After successful activation, you must wait 10
seconds before you can use the command again to transfer the premium status to another server.

If you want to activate premium on your server, consider checking out the many [Patreon Perks](/bot/premium-perks) you
will get!

## Command Structure

```
/activatepremium
```
