# Level Set

### Description

This command can be used to set a users level on demand.

The user you want to set a level to does not need to have prior levels.

If the user you want to set a level to does however have prior levels, their existing levels will be replaced with the new one.

If you want to add levels to a user that already has some, you can use the [level add](level-add.md) command

### Command Structure

```
/level set <user:> <amount:>
```
