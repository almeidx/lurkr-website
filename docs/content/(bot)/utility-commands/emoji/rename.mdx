### Description

<Callout type="warning">
	This is a **method** or **sub-command** of the [Emoji](./index) command. It is not its own command.
</Callout>

This method of the [emoji](./index) command allows you to rename an existing emoji in your server. You can provide either the emoji name or its ID to identify which emoji to rename.

### Command Structure

```
/emoji rename <emoji:> <name:>
```

| Option | Description                                    | Required |
| ------ | ---------------------------------------------- | -------- |
| emoji  | The name or ID of the emoji you want to rename | Yes      |
| name   | The new name for the emoji                     | Yes      |

### **Permission**

- `Manage Emojis` **(User)**
- `Manage Emojis` **(Bot)**
