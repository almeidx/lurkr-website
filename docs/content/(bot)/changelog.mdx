## Leveling Experience Configuration & User Privacy

6 November 2025

### Discord Bot [!toc]

- Added `/config set` option for **Leveling Experience Gain Interval**.
  - Allows you to configure how long Lurkr should wait before granting XP again.
  - This interval is per user.

- Added `/config set` option for **Leveling Experience Per Message**.
  - Allows you to configure the minimum & maximum XP granted per message.
  - The actual XP number is randomly selected between these two numbers.

- Added `/config toggle` option for **Gain Leveling Experience from Bot Commands**.
  - Allows you to choose whether or not slash command messages count as XP.

- Added `/privacy` command.
  - Setting your leaderboard privacy to true will anonymize you on all public Lurkr leaderboards!
  - This does not affect Members-only or Managers-only leaderboards!
  - It might take a few minutes for the effects to become visible!

- The originating server name is now displayed on level up messages sent in DMs.
- Fixed a bug where premium users with both Max & Ultimate were not assigned correctly.
- Fixed a bug where slash command messages were not counted as XP.
- Fixed a bug with the top leveling role assignment (Leaderboard Role).
- Fixed a bug where the top leveling user was not being updated properly.

### Dashboard [!toc]

- Added the new `/set` and `/toggle` commands into the Dashboard.
  - You can now change XP gain interval, XP gain per message and XP gain from slash commands.

- Added sorting buttons to the role rewards & multipliers section on the leveling page.
  - You can switch between ascending and descending order.

- Changed Server-Wide Default Customization to only accept hex colours.
- Fixed a bug causing selector buttons to not show their correct state.
- Fixed the global multiplier being counted towards the channel & role multiplier limit.

## Documentation Overhaul & Privacy Improvements

22 July 2025

### Discord Bot [!toc]

- Changed the default setting of the Web Leaderboard visibility to Members-only.
  - This is to improve privacy by not allowing all leaderboards to be indexed on the web by default.

- To comply with privacy laws, we have reset all visibility settings to Members-only!
  - If you want your Web Leaderboard to continue to be public, you must set it back to public.
  - To change the visibility go to your Lurkr Dashboard ‚Üí Leveling ‚Üí Web Leaderboard

- Changed the wording under the Web Leaderboard visibility options to be more transparent.

### Dashboard [!toc]

- Overhauled our Documentation to be prettier and more functional!
  - We'd welcome feedback if you use the documentation a lot, so please feel free to send suggestions!
  - We're still working on migrating the guides over to primarily use the Web Dashboard as a visual guide.

## Web Leaderboard Visibility & Leveling Speed

18 July 2025

### Discord Bot [!toc]

- Added visibility options for the web leaderboard!
  - Allows you to change if your leaderboard can be accessed by anyone, by server members or only by
    managers/administrators.
  - Allows you to change if your leaderboard will be indexed on the web for search engines.

- Added the ability to change your server's leveling speed!
  - Change the cooldown after gaining XP on a message (Default 1 minute)
  - The XP gained is a random number between an interval, you can now change this interval.
  - Be able to set a fixed XP gain!

- Increased the [Lurkr Ultimate](/../premium) Command Leaderboard Limit to 500!
  - Use `/leaderboard` on a Lurkr Ultimate enabled server to browse through the top 500 members!
  - Changed from previously 200.
  - Standard limit still 100.

- Fixed a bug that made users not be able to login to the dashboard sometimes.
- Embed level up messages are now recognized in the `/check` command.
- Fixed Lurkr informing users about 20% Leveling Boost when using `/vote` despite the multiplier being disabled on the
  server.

## Introducing the Lurkr API!

28 April 2025

### Discord Bot [!toc]

- Added a **public API** for other developers to take advantage of Lurkr's deep Leveling System for their own projects!
  - This can allow you to add leveling requirements for your own bots, add or remove levels\/xp based on other factors,
    and integrate Lurkr leveling into other games!
  - You can create and customize your API key on your [profile page](/../profile)
  - Please read the API Documentation before creating support requests: [lurkr.gg/docs/api](/api)

- Improved and updated the rank card!
  - Now applies the name on the rank card in this order: <u>1.</u> Nickname, <u>2.</u> Display name, <u>3.</u> Username
  - Removed the character limit for names on the rank card.
  - Removed the dark gray background around the Avatar in the rank card.

- Added the `level_or_milestone` option to the `/test` command.
  - Useful for checking how the level up message or milestone message will look like for specific levels or milestones.

- Fixed the rank card not showing when any image failed to load.
- Fixed cyrillic alphabet not appearing in the rank card.

### Dashboard [!toc]

- Added an API key generation section to the [profile page](/../profile)!

## Role Rewards placeholder on Level Up Messages

30 December 2024

### Discord Bot [!toc]

- Added `{roleRewards}` placeholder to Level Up Message!
  - This now allows you to show which role rewards your members get!
  - Also added support for showing text conditionally, using `{#var}text here{/var}` and `{^var}text here{/var}` for
    negative condition.
  - **For Example**: `{#roleRewards}You gained the {roleRewards} role rewards!{/roleRewards}`, this **only appears if**
    role rewards were applied.

- Added support for more colour names in the `/colour` command

- Fixed an issue with the handling of Polaris private leaderboard errors.
- Fixed only one mentioned role working at a time for mentioned role cooldowns.
- Minor stability improvements.

- Removed `/image`, `/purge`, and `/translate` commands.
  - They are no longer the focus of the bot, and weren't used by anyone anymore.

### Dashboard [!toc]

- Fixed a 404 error when accepting cookies in the dashboard page.
- Fixed roles not being saved when updating an existing role multiplier.

## Quality of Life Improvements

29 September 2024

### Discord Bot [!toc]

- Increased the Role Mention Cooldown interval to **6 Hours** for [Lurkr Ultimate](/../premium) servers.
  - The standard limit of 30 minutes has not been changed.

- Added a `user` option to `/background view` command.
  - This new option is meant to help users find and report (`/background report`) offensive user backgrounds.

- Added named colour options to the `/colour` command.
  - This means you can now type `red` or `green` instead of a complicated hex code! Give it a try!

- Fixed Polaris Import importing incorrect levels.
- Reduced probability of getting an "unexpected error" when using `/importxp`.
- Renamed the `until` option in the `/importxp` command to `min_level` for better clarity on what the option does.
- Fixed an issue not gaining XP when an active multiplier is too low (i.e., x0.01).

- Unfortunately Polaris Import's might still sometimes fail with "unexpected errors". This is because of Polaris'
  inherent instability and cannot be fixed on our end.

### Dashboard [!toc]

- Added a toggle for **Only Announce Level-Ups Together With Role Rewards** which was previously only available through
  the config command.
  - You can find it under **Leveling > Level Up Message**.

- Added showing the non-shortened number when hovering over it in the Leveling Calculator.
  - Instead of `21.3M`, hovering over the number with your mouse will show `21,303,511` now.

- Fixed level up message conditions not showing/updating correctly.
- Miscellaneous styling fixes.
