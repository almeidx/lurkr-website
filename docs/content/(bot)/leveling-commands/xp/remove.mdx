---
title: XP Remove
description: Remove XP from users.
---

### Description

This command can be used to remove a user's XP on demand.

The user you want to remove XP from **does** need to have prior XP.

The XP you specify in the command will be removed from their current XP.

### Command Structure

```
/xp remove <user:> <amount:>
```

### Permission

- `Manage Messages` **(User)**
- N/A **(Bot)**
